# 情绪周期量化指标体系

## 冰点状态量化标准

| 指标 | 阈值 | 信号强度 |
|------|------|----------|
| 跌停家数 | > 10个 | ⚠️ 强信号 |
| 下跌家数 | > 4500家 | ⚠️ 强信号 |
| 上涨家数占比 | < 10% | ⚠️ 强信号 |
| 涨停家数 | < 40家（28家极度低迷）| ⚠️ 强信号 |
| 连板高度 | 压制在4-5板 | ⚠️ 中等信号 |

**示例冰点数据**: 上涨867家 vs 下跌4541家（约15%:85%）

## 修复信号量化标准

| 指标 | 阈值 | 说明 |
|------|------|------|
| 封板率 | > 65% | 资金回流确认 |
| 涨停家数 | > 60家（63家起点）| 情绪回暖标志 |
| 上涨家数 | > 2800家 | 普涨格局形成 |
| 炸板率 | < 30% | 接力氛围改善 |

## 启动确认信号

| 指标 | 阈值 | 说明 |
|------|------|------|
| 二进三+三进四晋级率 | 100% | 情绪转强明确信号 |
| 竞价金额 | > 1.5亿 + 红开 | 资金抢筹标志 |
| 连板高度突破 | 换手5板破局 | 新周期开启标志 |

## 成交量阈值

| 成交量 | 意义 | 策略 |
|--------|------|------|
| 2万亿 | 结构性行情临界支撑值 | 低于此值谨慎操作 |
| 2.3万亿 | 足以支撑结构性行情 | 可积极参与 |
| 放量下杀V反 | 主跌结束信号 | 抄底机会 |

**缩量与放量辩证**:
- 底部位置: 缩量反弹不是坏事，放量反而危险（筹码松动）
- 顶部位置: 放量是出货特征，需警惕

## 炸板率阈值

| 炸板率 | 含义 | 应对策略 |
|--------|------|----------|
| < 30% | 正常/修复状态 | 积极参与 |
| ~30% | 震荡市/退潮初期 | 高抛低吸，控制仓位 |
| ~50% | 变盘日特征 | 非高手空仓观望 |

## 转换信号识别

### 冰点 → 启动
1. 成交量连续缩量后触及关键点位（如2万亿）后放量预期
2. 竞价金额达标（1.5亿+）+ 红开
3. 二进三+三进四晋级率100%
4. 指数跌到箱体底部（如4000点附近）

### 主升 → 退潮
1. 高位异常放量（弱势行情中是出货特征）
2. 炸板率飙升至50%
3. 情绪锚定股竞价不及预期
4. 高标补跌 + 大盘下杀共振

## 集合竞价锚点判断

1. **昨日最强板块辨识度作为锚定**
   - 超预期 → 考虑板块内辨识度个股套利
   - 符合预期 → 盘中观察回流情况
   - 不及预期 → 保留卖权，只卖不买

2. **关键信号**
   - 竞价撤单 + 撮合成交增多 = 场内场外分歧
   - 反核点位-3% = 资金跑路信号
