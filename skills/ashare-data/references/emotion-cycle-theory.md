# 情绪周期六阶段理论

## 核心模型

情绪周期本质是市场资金从冷静到狂热再到退潮的完整循环，分为六个标准阶段。

## 六阶段定义与特征

### 1. 冰点期
- **定义**: 市场亏钱效应最大化，连板高度压缩，资金观望
- **特征**:
  - 高位A杀，涨停家数少
  - 炸板率高，主线真空
  - 资金极度谨慎
- **策略**: 控制仓位，只做试错

### 2. 启动期
- **定义**: 资金开始试探进场，首板、二板出现
- **特征**:
  - 连板高度开始修复
  - 首板批量出现
  - 资金初步聚焦方向
- **策略**: 锁定主线，关注先锋

### 3. 发酵期
- **定义**: 资金形成初步共识，主线梯队完整
- **特征**:
  - 板块效应增强
  - 龙头加速
  - 量能放大
- **策略**: 聚焦龙头，大胆跟随

### 4. 主升期
- **定义**: 情绪进入一致狂热，主线全面爆发
- **特征**:
  - 批量涨停
  - 高位无恐慌
  - 资金只做主线
- **策略**: 重仓干，持有核心

### 5. 分歧期
- **定义**: 主线内部分化，高位震荡
- **特征**:
  - 龙头高位换手
  - 炸板增多
  - 梯队分离，强弱分化
- **策略**: 去弱留强，拥抱穿越

### 6. 退潮期
- **定义**: 情绪快速冷却，高位杀跌
- **特征**:
  - 龙头见顶A杀
  - 补涨龙崩
  - 板块无接力
- **策略**: 空仓观望，不做反弹

## 操作策略汇总表

| 阶段 | 操作策略 | 核心要点 |
|------|----------|----------|
| 冰点期 | 控制仓位，只做试错 | 不激进，等待新主线酝酿 |
| 启动期 | 锁定主线，关注先锋 | 识别主线雏形，参与先锋标的 |
| 发酵期 | 聚焦龙头，大胆跟随 | 板块效应增强，拥抱核心 |
| 主升期 | 重仓干，持有核心 | 资金只做主线，吃满利润 |
| 分歧期 | 去弱留强，拥抱穿越 | 高位换手，筛选真正强者 |
| 退潮期 | 空仓观望，不做反弹 | 规避风险，等待下一次开启 |

## 主线五浪与情绪六阶段对应关系

| 五浪阶段 | 对应情绪阶段 | 说明 |
|----------|--------------|------|
| 一浪（启动试错）| 冰点→启动 | 主线雏形出现 |
| 二浪（弱分歧/确认）| 发酵期弱洗盘 | 确认主线有效性 |
| 三浪（主升爆发）| 情绪主升期 | 最暴利阶段 |
| 四浪（大分歧/切换）| 情绪分歧期 | 新旧龙交替 |
| 五浪（末浪冲高→退潮）| 分歧末期→退潮 | 周期结束 |

> **核心观点**: 五浪是空间结构（K线怎么走），六阶段是时间灵魂（资金情绪怎么演变）。两者合一，才是完整的主线周期体系。

## 量化时代龙头新定义

- **真龙特征**: 从死人堆里爬出来，市场恐慌时从未被按死
- **龙头功能**: 扭转市场惯性，提示市场此处摁不死了
- **风格轮动**: 连板与趋势风格不停轮动，各有节奏
